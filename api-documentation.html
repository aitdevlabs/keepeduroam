Overview

KeepEduroam is a WebSocket relay service that connects providers (users sharing internet) with consumers (users needing internet access). This API documentation covers how clients can integrate with the service.

🌐 Base URL

https://keepeduroam.aitdevlabs.workers.dev

📡 Protocol

Primary: WebSocket (Socket.IO v4)
Fallback: HTTP long-polling
SSL/TLS: Enabled (Cloudflare)

📦 Response Format

JSON

🔒 CORS

All endpoints support CORS

🔒 Admin Dashboard: The admin interface is hidden behind a secret path and is not accessible via the public API.

Events

Client → Server Events

EventPayloadDescription
register_provider {id: string} Register as a provider (share internet)
register_consumer {id: string} Register as a consumer (use internet)
consumer_request {target, port, payload} Request a relay from a provider
relay_response {id, result} Provider returns response
points_update {points: int} Update provider's points
ping {} Keep connection alive

Server → Client Events

EventPayloadDescription
connected {status, sid} Connection confirmed
registered {status, role, id} Registration confirmed
update_providers {sid: {id, points, ...}} Provider list update
update_consumers {sid: {id, data_used, ...}} Consumer list update
relay_request {id, target, port, payload} Relay request for provider
relay_result {id, result} Relay response for consumer
points_updated {points: int} Points updated
pong {timestamp: int} Ping response

→ register_provider
Register a device to share internet connection
socket.emit('register_provider', {
    "id": "device_001"
});

{
    "status": "ok",
    "role": "provider",
    "id": "device_001"

2. Register as Consumer

→ register_consumer

    "id": "device_002"

{
    "status": "ok",
    "role": "consumer",

3. Relay Request

Important: Consumer must be registered before sending relay requests.
→ consumer_request Consumer
Request a relay through an available provider
socket.emit('consumer_request', {
    "target": "example.com",
    "port": 80,
    "payload": "GET / HTTP/1.1\r\nHost: example.com\r\n\r\n"
});
→ relay_request Provider
Provider receives the relay request
{
    "id": "173...",
    "target": "example.com",
    "port": 80,
    "payload": "GET / HTTP/1.1\r\nHost: example.com\r\n\r\n"
}
← relay_response Provider → Server
Provider returns the response
socket.emit('relay_response', {
    "id": "173...",
    "result": "HTTP/1.1 200 OK\r\n..."
});
← relay_result Server → Consumer
Consumer receives the relay response

Integration Examples

Python Client

# Complete KeepEduroam client in Python
import socketio
import socket

class KeepEduroamClient:
    def __init__(self, mode='provider', device_id='device_001'):
        self.server = 'https://keepeduroam.aitdevlabs.workers.dev'
        self.mode = mode
        self.device_id = device_id
        self.sio = socketio.Client()
        self.setup_handlers()

    def setup_handlers(self):
        @self.sio.event
        def connect():
            print('✅ Connected to KeepEduroam')
            self.sio.emit(f'register_{self.mode}', {'id': self.device_id})

        @self.sio.event
        def registered(data):
            print(f'✅ Registered: {data}')

        @self.sio.event
        def update_providers(data):
            print(f'📊 Providers online: {len(data)}')

        @self.sio.event
        def relay_request(data):
            if self.mode == 'provider':
                result = self.forward_request(data)
                self.sio.emit('relay_response', {
                    'id': data['id'],
                    'result': result
                })

    def forward_request(self, data):
        sock = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
        sock.settimeout(10)
        sock.connect((data['target'], data['port']))
        sock.send(data['payload'].encode())
        response = sock.recv(4096)
        sock.close()
        return response.decode('utf-8', errors='ignore')

    def connect(self):
        self.sio.connect(self.server, transports=['websocket'])
        self.sio.wait()

# Usage
client = KeepEduroamClient(mode='provider', device_id='phone_001')
client.connect()

JavaScript Client

// Complete KeepEduroam client in JavaScript
import io from 'socket.io-client';

class KeepEduroamClient {
    constructor(mode = 'provider', deviceId = 'device_001') {
        this.server = 'https://keepeduroam.aitdevlabs.workers.dev';
        this.mode = mode;
        this.deviceId = deviceId;
        this.socket = null;
    }

    setupSocket() {
        this.socket = io(this.server, { transports: ['websocket'] });

        this.socket.on('connect', () => {
            console.log('✅ Connected to KeepEduroam');
            this.socket.emit(`register_${this.mode}`, { id: this.deviceId });
        });

        this.socket.on('registered', (data) => {
            console.log('✅ Registered:', data);
        });

        this.socket.on('relay_request', async (data) => {
            if (this.mode === 'provider') {
                try {
                    const res = await fetch(`http://${data.target}:${data.port}`, {
                        method: 'POST',
                        body: data.payload
                    });
                    const result = await res.text();
                    this.socket.emit('relay_response', {
                        id: data.id,
                        result: result
                    });
                } catch (error) {
                    this.socket.emit('relay_response', {
                        id: data.id,
                        result: `Error: ${error.message}`
                    });
                }
            }
        });
    }

    connect() {
        this.setupSocket();
        this.socket.connect();
    }
}

// Usage
const client = new KeepEduroamClient('provider', 'phone_001');
client.connect();

Error Handling

CodeMessageSolution
400 Not registered as consumer Call register_consumer first
400 Not registered as provider Call register_provider first
404 No providers available No providers online, wait
503 Service unavailable Server is down, retry later
Error handling example: 
socket.on('error', (data) => {
    console.error('❌ Error:', data.message);
    if (data.message === 'No providers available') {
        // Retry after delay
        setTimeout(() => reconnect(), 5000);
    }
});
